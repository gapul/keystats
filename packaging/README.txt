Keystats — macOS 打鍵アナリティクス
====================================

「どのキーを・どのアプリで・いつ(時間単位)」叩いたかを記録して、
キーボードのヒートマップや時間帯別の統計を見られるツールです。

★ プライバシー
   入力したテキストの本文は一切保存しません。
   記録するのはキーコード(どのキーか)とアプリ名と時刻だけです。


== はじめかた ==

   1. 「Keystats」を「アプリケーション」フォルダへ移動する
   2. 「Keystats」をダブルクリックして開く
   3. 画面の「入力監視の設定を開く」を押す
   4. システム設定で「Keystats」をオンにする
   5. Keystatsへ戻り「ダッシュボードを見る」を押す

   入力監視がオンになったことは自動で確認されます。
   黒い画面での操作や、コマンドの入力は必要ありません。


== アンインストール ==

   「Keystatsをアンインストール」を 右クリック →「開く」
   (記録データは残ります)


== 使い方 ==

- メニューバーのキーアイコン: 今日の打鍵数を常時表示
- クリック →「ダッシュボードを開く」: ヒートマップ / 時間帯別 /
  日別トレンド / よく押すキー / ショートカット / アプリ別
- アップデートは自動で確認します(メニューでオフにもできます)


対応: Apple Silicon Mac / macOS 13 以降
