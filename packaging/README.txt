keystats — macOS 打鍵アナリティクス
====================================

「どのキーを・どのアプリで・いつ(時間単位)」叩いたかを記録して、
キーボードのヒートマップや時間帯別の統計を見られるツールです。

★ プライバシー: 入力したテキストの本文は一切保存しません。
   記録するのはキーコード(どのキーか)とアプリ名と時刻だけです。


== インストール ==

1. install.command を「右クリック → 開く」→「開く」
   (ダウンロードしたアプリなので初回だけこの手順が要ります)

2. 途中で「システム設定 > プライバシーとセキュリティ > 入力監視」が開きます。
   一覧の "Keystats" のトグルをオンにしてください。

3. これで完了。メニューバーにキーのアイコンと今日の打鍵数が出ます。
   アイコンをクリック → 「ダッシュボードを開く」で詳しい統計が見られます。


== 使い方 ==

- メニューバーアイコン: 今日の打鍵数を常時表示
- ダッシュボード: キーボードヒートマップ / 時間帯別 / 日別トレンド /
  よく押すキー / ショートカット / アプリ別
- アップデート: 自動で確認します。メニューの「アップデートを確認」でも可能


== アンインストール ==

uninstall.command を実行してください(データは残ります)。


== 対応環境 ==

Apple Silicon Mac / macOS 13 以降。
